# ruin
1. ruin（动词）—— 破坏、毁坏（最常见用法）
核心同义词（高频、通用）：[[destroy]] [[wreck]] [[demolish]] [[devastate]] [[ravage]] [[smash]] [[crush]] [[spoil]] [[damage]] [[ruin]] (本身最中性)

更严重的毁灭 / 彻底毁掉：
[[annihilate]] [[obliterate]]
lay waste (to)
[[raze]] [[torpedo]] [[undo]] [[dash]] (hopes)

经济 / 个人层面毁掉（破产、毁前途）：
[[bankrupt]] [[bust]] [[break]] [[impoverish]]
wipe out
clean out
bring down

轻度 / 搞砸 / 糟蹋：
[[spoil]] [[mess]] up
[[screw]] up
[[botch]] [[mar]]
blight

常见搭配：
ruin someone's life / career / reputation（毁掉人生/事业/名声）
ruin the surprise / moment / mood（毁掉惊喜/气氛）
ruin a relationship / friendship（毁掉关系）
ruin the economy / country（毁掉经济/国家）
ruin everything（全毁了）

2. ruin（名词）—— 毁灭、废墟、祸根
常见同义词：
[[destruction]] [[devastation]] [[ruination]] [[downfall]] [[collapse]] [[undoing]] [[bane]] [[havoc]] [[wreck]] [[demise]]

废墟 / 残骸（古建筑等）：
ruins (复数最常见)
[[remnant]]s
[[wreckage]] [[rubble]] [[debris]] [[relics]] [[vestiges]]

比喻 / 导致毁灭的原因：
[[bane]] [[curse]] [[scourge]] [[nemesis]] [[waterloo]] (比喻致命失败)

形容词形式 & 相关状态：
[[ruined]]（毁坏的、破产的）
[[ruinous]]（毁灭性的、代价高昂的）
[[dilapidated]]（破败的）
[[crumbling]] [[derelict]] [[decrepit]]
in ruins（一片废墟）

3. 反义词（antonyms）—— 修复、建设、成功
[[build]] [[construct]] [[repair]] [[restore]] [[fix]] [[improve]] [[salvage]] [[recover]] [[rebuild]] [[enrich]] [[save]]

4. 记忆小联想链
物理破坏 → [[wreck]] / [[demolish]] / [[raze]]（像推倒大楼）
彻底毁灭 → [[devastate]] / [[annihilate]] / lay waste（像战争/灾难后）
经济破产 → [[bankrupt]] / [[bust]] / wipe out（像股市崩盘）
搞砸机会 → [[spoil]] / [[ruin]] / [[blow]]（口语：You just ruined it!）
废墟画面 → [[ruins]] / [[rubble]] / remnants（古罗马废墟、游戏里的ruins）
导致失败的根源 → [[bane]] / [[downfall]] / [[undoing]]（His pride was his ruin.）
