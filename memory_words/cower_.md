# cower
cower 这个词是动词，意思是因恐惧而蜷缩、畏缩、退缩、发抖（通常是身体动作，表现极度害怕或顺从，常带画面感）。
它常用于人、动物在危险/权威/威胁面前的反应，比 fear 或 be afraid 更具体、更生动。
下面按用法给你整理同义词、强联想词和地道表达。
1. cower（动词）—— 核心同义词（按强度/语境排序）
最常用 / 高频替换（最接近、口语到正式）：

cringe（最常用！畏缩、退缩，常带尴尬或恐惧）
shrink (back / away)（向后缩、退缩）
recoil（猛地后退、畏缩）
flinch（畏缩、退缩，常因疼痛或惊吓）
quail（畏缩、胆怯，较正式/文学）
huddle (down)（蜷缩成一团）
crouch (in fear)（蹲下、蜷缩）

更强 / 更形象的同义：

tremble / shake（发抖）
cower in terror / submission（因恐惧/顺从而畏缩）
grovel（卑躬屈膝，更低三下四）
slink away（偷偷溜走，带害怕感）

反义词（antonyms）：

stand tall / stand up to
confront / face
defy
brazen it out（厚着脸皮面对）

2. 强联想词 / 常见搭配（collocations，高频真实用法）

cower in fear / terror / dread（因恐惧而畏缩）
cower in the corner / corner of the room（蜷缩在角落）
cower before / from sb / sth（在某人/某事面前畏缩）
cower away / back（向后畏缩）
cowering child / dog / animal（畏缩的孩子/狗/动物）
cower under the table / bed / blanket（躲在桌子/床/毯子下发抖）
make sb cower（让某人畏缩）
cower in submission（顺从地畏缩）

3. 地道表达 & 习语 / 真实场景用法

cower in fear（最经典、最自然的搭配）
e.g. The dog cowered in fear when it heard the thunder.（听到雷声，那狗吓得蜷缩成一团。）
cower before someone（在权威/强权面前畏缩）
e.g. He always cowers before his boss.（他在老板面前总是畏畏缩缩的。）
make someone cower（让某人吓得退缩）
e.g. His angry shout made the kids cower.（他一声怒吼把孩子们吓得直往后缩。）
cower in the shadows / corner（躲在暗处/角落里发抖）
e.g. She cowered in the shadows, hoping no one would see her.（她蜷缩在阴影里，希望没人发现她。）
cower like a beaten dog（像挨打的狗一样畏缩 → 超级形象的比喻）
e.g. He cowered like a beaten dog when his father yelled at him.（父亲一吼，他就吓得像条挨打的狗。）
still cowering（还在畏缩中，常用于描述余惊未消）
e.g. Even after the storm passed, the animals were still cowering.（风暴过去了，动物们还在发抖。）
口语调侃 / 夸张版：
“Stop cowering! Stand up for yourself!”（别再畏畏缩缩了！硬气点！）

4. 记忆小联想链（超级好记）

画面感 → 狗/小孩蜷缩在角落 → cower in the corner / like a beaten dog
动作 → 身体向后缩、发抖 → cringe / shrink / flinch / recoil
场景 → 面对老板/雷声/怪物 → cower before / in fear / in terror
情绪升级 → fear → tremble → cower（最强烈的身体畏缩反应）
对比 → stand tall（挺直腰杆） vs cower（缩成一团）

一句话总结：
cower = “吓得缩成一团”的生动画面词，比单纯的“害怕”多了身体动作和视觉感，常出现在小说、电影、新闻（描述弱势群体或动物时）。
