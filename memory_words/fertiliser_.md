# fertiliser


fertiliser（英式拼写，美式为 fertilizer）意思是“肥料”，指用于增加土壤肥力、促进植物生长的物质（如有机肥或化学肥）。

1. 同义词（Synonyms）
常见同义词（从自然到人工）：

manure：粪肥、畜肥（最常见，尤其是动物粪便）
compost：堆肥（有机废料发酵而成）
plant food：植物养料、植物肥
dressing / top dressing：追肥、表肥
dung：（动物）粪
muck：（湿的）粪肥、腐殖质
mulch：覆盖物（有时兼有保湿和施肥作用）
humus：腐殖质
guano：海鸟粪肥（或蝙蝠粪肥）
bone meal：骨粉
potash：钾肥
urea：尿素（常见化学肥料成分）
其他区分：

organic fertilizer / natural fertilizer：有机肥
chemical fertilizer / artificial fertilizer / synthetic fertilizer：化学肥、合成肥
inorganic fertilizer：无机肥
反义词（antonyms）通常是 pesticide（杀虫剂）、herbicide（除草剂）等农药类。

2. 联想词 / 相关词（Associated / Related Words）
soil（土壤）、nutrient（养分）、nitrogen（氮）、phosphorus（磷）、potassium（钾，简称 NPK）
ammonium nitrate（硝酸铵）、phosphate（磷酸盐）
overuse / runoff（肥料流失，导致水体富营养化 eutrophication）
application / spread（施肥）
yield（产量）、crop（作物）
相关动作：fertilize（施肥）、fertilization（施肥过程）
常见搭配（collocations）：

chemical fertilizer / artificial fertilizer
apply / spread / use fertilizer
amount of fertilizer
nitrogen fertilizer
organic vs chemical fertilizer
3. 地道表达（Natural Expressions）
Add some fertilizer to the soil.（给土壤加点肥料。）
The plants need fertilizing.（这些植物需要施肥了。）
Over-application of fertilizer can harm the environment.（过度施肥会损害环境。）
We're switching to organic fertilizer this year.（我们今年改用有机肥。）
This compost makes excellent natural fertilizer.（这种堆肥是很好的天然肥料。）
The field was heavily fertilized.（这片田施了很多肥。）
Fertilizer runoff pollutes rivers.（肥料径流污染河流。）
在农业或园艺语境中，常说 “feed the soil, not the plant”（养土而非直接喂植物），强调有机肥的重要性。

4. 发音（Pronunciation）
英式（fertiliser）：/ˈfɜː.tɪ.laɪ.zə(r)/
大致读作 “FUR-ti-lye-zer” （重音在第一个音节，-iser 读 /aɪ.zə/）
美式（fertilizer）：/ˈfɝː.t̬ə.laɪ.zɚ/
大致读作 “FURR-duh-lye-zer” （r 音明显）
听起来像 “fertile” + “-izer”。注意不要把重音放在第二个音节（常见错误）。

5. 拼写易混词 & 注意事项
英美拼写差异：
英式/澳式/加式常用 fertiliser（用 s）；
美式常用 fertilizer（用 z）。
动词形式同理：fertilise（英） vs fertilize（美）。
易混淆词：
fertile（形容词：肥沃的、能生育的）
fertility（名词：肥力、生育力）
fertilization（受精或施肥过程）
fossilizer / fossiliser（化石化，不相关，但拼写有点像）
常见拼写错误：把 “fertiliser” 写成 “fertliser”（漏 i）、“fertilizor” 或混淆 s/z。
