# condom

condom（名词）指“避孕套、安全套”。
这是一个直接、正式的医学/日常用词。
1. 同义词（Synonyms）
正式 / 中性：

contraceptive sheath（避孕套，较正式）
prophylactic（避孕套，医学用语，尤其美式）
rubber（橡胶套，旧式/口语）
latex condom（乳胶避孕套）
male condom（男用避孕套，与 female condom 区分）

俚语 / 口语（最常用）：

rubber（最常见美式俚语）
johnny（英式、澳式俚语）
jimmy / jimmy hat（美式俚语）
love glove（幽默说法）
raincoat（幽默：雨衣）
French letter（旧式英式俚语）
safe / safety（较少用）
wrapper（包装，俚语）
trojan（品牌代称，像 Kleenex 指纸巾）

其他类型：

female condom（女用避孕套，也叫 FC2）
dental dam（牙坝，用于口交防护，非避孕套）

反义词（antonyms）：bareback（无套性交）、raw（无保护措施）。
2. 联想词 / 相关词（Associated / Related Words）

contraception（避孕）、birth control（节育）
protection（保护）
safe sex（安全性行为）
STI / STD（性传播感染 / 疾病）
HIV prevention（艾滋病预防）
lube（润滑剂）
spermicide（杀精剂）
condom wrapper（避孕套包装）
expiration date（有效期）
break / tear / slip（破裂 / 滑脱）
相关动作：put on / roll on / wear a condom（戴避孕套）

常见搭配：

use / wear a condom
put on a condom
condom use
break the condom
buy / carry condoms
free condoms（免费避孕套，常在诊所或活动发放）

3. 地道表达（Natural Expressions）

Always use a condom to protect against STIs.（一定要使用避孕套来预防性传播感染。）
Did you bring any condoms?（你带避孕套了吗？）
Make sure you put the condom on correctly.（确保正确戴上避孕套。）
We only have sex with condoms.（我们只用避孕套做爱。）
He forgot to wear a condom.（他忘了戴避孕套。）
I always carry a couple of rubbers just in case.（我总是随身带几个避孕套以防万一。）
The condom broke — we need to get the morning-after pill.（避孕套破了，我们得去买紧急避孕药。）
She insisted on using a condom.（她坚持要用避孕套。）
Free condoms are available at the clinic.（诊所提供免费避孕套。）
No glove, no love.（不戴套，就别做。——常见幽默口号）

在口语中，美式英语常用 “rubber”，英式常用 “johnny”。年轻人之间常直接说 “condom” 或 “protection”。
4. 发音（Pronunciation）

英式（British）：/ˈkɒn.dɒm/
读作 “KON-dom”（重音在第一个音节，两个音节都很清楚）。
美式（American）：/ˈkɑːn.dəm/
读作 “KAHN-dum”（第一个音节较长，第二个音节弱读成 /dəm/）。

注意：不要把重音放在第二个音节（常见错误），也不要读成 “con-DOM”。
5. 拼写易混词 & 注意事项

无英美拼写差异：condom 在英式和美式中拼写完全相同。
最常见混淆：
condom vs condemn（谴责，发音完全不同 /kənˈdem/）
condom vs kingdom（王国）
condom vs random（随机的）
condom vs condone（宽恕）


常见拼写错误：把 “condom” 写成 “condum”（漏 o）或 “con-dom”（加连字符，不正确）。
文化小贴士：
“condom” 是最直接、中性的词，在医疗、性教育和日常对话中都合适。俚语（如 rubber / johnny）在朋友间更自然，但在正式场合或与长辈对话时最好用 “condom”。不同国家和文化对这个话题的开放程度差异很大，使用时要注意场合。
与之前学的词对比：

virginity（童贞状态）
paragliding（滑翔伞运动）
condom 属于性健康 / 避孕相关词汇，常与责任、安全、预防联想。
