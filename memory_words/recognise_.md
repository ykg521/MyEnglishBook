# recognise
recognise 完整单词笔记
英 /ˈrekəɡnaɪz/　美 /ˈrekəɡnaɪz/
（美式拼写：recognize）
中文：认出；承认；意识到；认可
一、同义词（按含义分组）
1）认出、辨认（视觉 / 身份）
identify 识别，确认身份
spot 一眼认出，发现
pick out 辨认出，挑出
distinguish 区分，辨别
know 认识，认得
place （口语）想起是谁 / 在哪见过
2）承认、认可（事实 / 地位 / 成就）
acknowledge 承认；致谢
admit 承认（事实、错误）
concede 勉强承认
accept 接受，认可
grant 承认，赋予
allow 承认，认可
3）意识到、觉察到
realize 意识到
appreciate 领会，意识到
perceive 察觉，理解
detect 察觉，发现
二、联想词 & 派生词
recognition n. 认出；承认；认可
recognizable adj. 可辨认的
recognizable voice/face 熟悉的声音 / 面孔
unrecognizable adj. 认不出的，面目全非的
identify 识别
identification 身份证明
acknowledgment 承认；致谢
awareness 意识
notice 注意到
realization 领悟，认识
三、地道搭配 & 高频表达
1）认出
recognise sb by sight/voice 凭外貌 / 声音认出某人
hardly recognise sb 几乎认不出某人
recognise an old friend 认出一位老朋友
2）承认、认可
recognise a problem/fact 承认问题 / 事实
recognise one’s mistake 承认错误
recognise the need for… 认识到…… 的必要性
recognise authority 认可权威
officially recognise 正式承认
3）赏识、表彰
in recognition of 为表彰……；作为对…… 的认可
gain/win recognition 获得认可 / 赞誉
deserve recognition 值得肯定
四、发音、拼写易混词
1）拼写易混
reconcile /ˈrekənsaɪl/
调和；和解；使一致（拼写结构非常像）
recollect /ˌrekəˈlekt/
回忆，想起
cognise / cognize
认知，知道（同词根，易混）
recommend /ˌrekəˈmend/
推荐（开头 re- 长单词易混）
2）发音易混
recognize / recognise 英美拼法不同，发音一致
organise / organize /ˈɔːɡənaɪz/
结构相同：-ise/-ize 结尾，重音位置相似
realize / realise /ˈriːəlaɪz/
词义接近、后缀相同，极易混用
五、小辨析
recognise：强调再次认出或正式认可
realize：强调心里意识到
acknowledge：更正式，侧重公开承认
admit：常指不情愿承认（错误、事实）
