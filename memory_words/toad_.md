# toad

toad 是可数名词，主要指“蟾蜍”（一种皮肤干燥、疙瘩状、与青蛙相似的两栖动物，常生活在陆地附近）。

1. 同义词（Synonyms）
动物含义（字面）：

frog（青蛙，最接近，常被混用）
amphibian（两栖动物）
batrachian / anuran / salientian（科学术语，较正式）
bullfrog（牛蛙，特定种类）
俚语 / 比喻含义（讨厌的人，贬义）：

creep（令人不适的人）
jerk / bastard / dog / rat / skunk / snake / swine / louse / heel / scumbag（混蛋、卑鄙小人）
beast / reptile（禽兽、爬虫，比喻讨厌鬼）
toad（本身就是俚语中的贬义词）
反义词（antonyms，主要针对贬义）：hero、gentleman、lady、saint、angel（英雄、绅士、淑女、圣人、天使）。

2. 联想词 / 相关词（Associated / Related Words）
toadstool（伞状毒蘑菇，经典联想：因为古人认为蟾蜍有毒，蘑菇像蟾蜍坐的凳子）
warty（疙瘩状的）、slimy（黏滑的）、dry skin（干燥皮肤，与 frog 的湿滑皮肤对比）
croak（蛙/蟾的叫声）、hop（跳跃）、tadpole（蝌蚪）
garden（花园，常有蟾蜍吃害虫）、pond（池塘）、moist habitat（潮湿栖息地）
hallucinogenic toad（某些蟾蜍分泌物有致幻成分，如 Colorado River toad）
文化联想：ugly（丑陋）、poisonous（有毒的）、fairy tale（童话，如公主吻蟾蜍变王子）
常见搭配（collocations）：

a slimy / warty / brown / green toad
hear a toad croaking
toad in the garden / pond
toadstool（毒蘑菇）
ugly as a toad
3. 地道表达（Natural Expressions）
字面含义：

I almost stepped on a toad in the garden last night.（昨晚我在花园差点踩到一只蟾蜍。）
Toads eat insects and are good for the garden.（蟾蜍吃虫，对花园有益。）
The toad hopped away when I approached.（我靠近时，那只蟾蜍跳走了。）
We found some toadstools under the tree after the rain.（雨后我们在树下发现了一些毒蘑菇。）
比喻 / 俚语含义（讨厌的人）：

He's such a nasty little toad!（他真是个讨厌的小混蛋！）
Don't listen to that toad — he's always lying.（别听那个混蛋的话，他老是撒谎。）
You arrogant toad!（你这个傲慢的家伙！）
其他地道用法：

Toad in the hole（英式传统菜：香肠烤在约克郡布丁里，字面“洞里的蟾蜍”）
Frog and toad（澳式俚语，rhyming slang，指“road”道路，如 “hit the frog and toad” = 上路）
在童话或故事中常说 “The princess kissed the toad and it turned into a prince.”（公主吻了蟾蜍，它变成了王子。）

4. 发音（Pronunciation）
英式（British）：/təʊd/
大致读作 “tohd”（长元音 /əʊ/，像 “go” 或 “show” 中的 o 音）。
美式（American）：/toʊd/
大致读作 “tohd”（同样长元音 /oʊ/）。
一个音节，重音在唯一音节上。听起来像 “towed”（拖车）或 “toed”（有脚趾的）——这些是同音词。

5. 拼写易混词 & 注意事项
最常见的混淆：

toad /toʊd/（蟾蜍）
towed /toʊd/（被拖走的，如车被拖）
toed /toʊd/（有……脚趾的，如 three-toed sloth 三趾树懒）
toadstool（毒蘑菇，常被误以为是可食用的 mushroom）
其他易混：

toad vs todd（人名 Todd，发音有时接近）
toad vs trod（tread 的过去式 /trɒd/）
小贴士：
toad 在文学或童话中常象征“丑陋但可能变好”，而在口语中多为贬义骂人词（比 frog 更负面，因为蟾蜍皮肤干燥多疙瘩，看起来更“恶心”）。toadstool 是固定词，不要分开写成 “toad stool”。

与之前学的词对比：

creep（偷偷移动 / 令人不适的人）—— toad 作为贬义时类似 creep。
wail、volatile 等无关，这里 toad 属于动物 + 俚语贬义词汇，常与花园、童话、毒蘑菇联想。
