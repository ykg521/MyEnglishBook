# entice
1. entice（动词）—— 核心同义词（按强度排序）
最强 / 最直接（强烈诱惑、勾引感）：
[[lure]]（最常用，带“用饵引诱”感觉）
[[tempt]]（诱惑，常用于食物、罪恶、机会）
[[seduce]]（勾引、诱惑，常带性感或背叛意味）
[[allure]]（魅力吸引、迷人）

中等强度（巧妙说服、甜言蜜语）：
[[beguile]]（用魅力迷惑、哄骗）
[[cajole]]（哄骗、甜言哄）
[[coax]]（哄、劝诱，常用于温和场景）
[[inveigle]]（花言巧语诱使，稍正式/负面）
[[wheedle]]（甜言哄、撒娇式诱导）

其他常见近义：
[[attract]]（吸引，更中性）
[[draw]]（拉过来、吸引）
[[bait]]（用饵引诱）
[[decoy]]（设饵诱捕）
[[entice]] away（引诱离开）
[[lead]] on（引诱、吊胃口）

2. 常见搭配 / 强联想短语（collocations & phrases）
这些是entice最常出现的真实用法，超级实用：
entice someone to do sth（诱使某人做某事）
entice customers to buy（吸引顾客购买）
entice investors（吸引投资者）
entice tourists / visitors（吸引游客）
entice buyers / shoppers（吸引买家）
entice away（引诱离开）
entice talent away from competitors（挖角人才）
entice with（用…诱惑）
entice with promises / offers / discounts（用承诺/优惠诱惑）
entice with food / smell / beauty（用食物/香味/美貌吸引）
be enticed by / into（被…诱惑）
be enticed by the offer（被优惠吸引）
enticed into a trap（被诱入陷阱）

3. 形容词 / 相关状态词
[[enticing]]（诱人的、迷人的）
an enticing offer / smell / prospect（诱人的报价/香味/前景）
[[enticement]]（诱惑物、诱饵）—— 名词形式
[[seductive]]（诱惑的、勾人的）
[[tempting]]（诱人的、很吸引人）
[[alluring]]（迷人的、有魅力的）

4. 反义词（antonyms）—— 排斥、警告
[[repel]]（排斥、赶走）
[[deter]]（阻止、劝阻）
[[discourage]]（使气馁、劝阻）
warn off / drive away（赶走、警告离开）

记忆小联想链
钓鱼 → [[lure]] / [[bait]]（用饵引鱼上钩）
美食广告 → [[tempt]] / [[entice]]（闻到香味忍不住）
恋爱 / 暧昧 → seduce / allure / beguile（用魅力勾引）
商场促销 → entice with discounts（用打折吸引顾客）
负面陷阱 → lure into a trap / inveigle（诱入圈套）
