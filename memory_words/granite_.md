# granite

[[granite]] 这个词在英语里有两种主要含义：

    名词：花岗岩（一种硬质火成岩，常用于建筑、台面、纪念碑等）
    比喻 / 形容词用法：坚如磐石、顽强、不屈不挠（指性格、意志、决心等非常坚硬、不可动摇，常带正面或中性意味）

下面按含义给你整理同义词（synonyms）、强联想词（associated words / collocations）和地道表达（idiomatic / natural expressions），方便记忆和地道使用。
1. granite（名词）—— 花岗岩（岩石）

核心同义词（地质/建筑语境）：

    rock（广义石头）
    stone（石头，更通用）
    igneous rock（火成岩，科学用）
    plutonic rock（深成岩，花岗岩属于这一类）
    hard rock（硬岩石）

其他常见类似岩石（常一起比较）：

    marble（大理岩，常跟granite对比台面）
    quartzite（石英岩）
    sandstone（砂岩）
    limestone（石灰岩）
    basalt（玄武岩）
    slate（板岩）
    gneiss（片麻岩）

强联想词 / 常见搭配（collocations，高频真实用法）：

    granite counter / countertop / kitchen counter（花岗岩台面 → 最常见家居用法）
    granite slab / block / boulder（花岗岩板/块/巨石）
    granite monument / tombstone / gravestone（花岗岩纪念碑/墓碑）
    granite quarry（花岗岩采石场）
    polished granite（抛光花岗岩）
    black granite / pink granite / red granite / gray granite（黑/粉/红/灰花岗岩）
    granite steps / stairs / cladding / facade（花岗岩台阶/外墙）

地道表达：

    Granite is durable and heat-resistant, perfect for kitchen counters.（花岗岩耐用耐热，很适合做厨房台面。）
    The countertop is made of solid granite — it won't scratch easily.（台面是纯花岗岩做的——不容易刮花。）
    They chose black granite for the gravestone.（他们选了黑花岗岩做墓碑。）
    Granite City（花岗岩城 → 美国有些城市的昵称，如 Aberdeen, Scotland 或某些美国城市）

2. granite（比喻用法）—— 坚如磐石、不屈的意志 / 决心

核心同义词（性格/意志语境，最常见比喻）：

    resolve
    determination
    firmness
    steadfastness
    iron will / ironclad determination
    steel（钢铁意志）
    rock-solid
    unyielding
    inflexible
    stubbornness（顽固，带负面）

更强的同义：

    granite-like resolve / will
    backbone of steel
    unshakeable / unbreakable

地道表达 & 常见习语 / 比喻：

    a will of granite / heart of granite（花岗岩般的意志/心肠 → 非常坚韧或冷酷）
    e.g. She has a will of granite — nothing can stop her.（她意志如磐石，什么都挡不住她。）
    carved / etched / set in granite（像刻在花岗岩上一样不可改变）
    e.g. His opinion is set in granite; he won't budge.（他的观点坚如磐石，不会动摇。）
    （类似 carved in stone，但 granite 更强调硬度）
    face like granite / expression of granite（面无表情、冷硬如花岗岩）
    e.g. He stared back with a face like granite.（他回瞪着，脸冷硬如花岗岩。）
    take it for granite（常见拼写错误，误写成 take it for granted“想当然”）
    → 很多人打错成“take it for granite”，成了网络梗，但正确是 granted。
    granite-jawed / granite-faced（下巴/脸像花岗岩一样坚硬 → 形容硬汉形象）
    e.g. The granite-jawed detective refused to back down.（下巴硬如花岗岩的侦探不肯退让。）

记忆小联想链

    物理硬岩 → granite counter / slab / monument → marble / quartzite / stone
    厨房装修 → polished granite countertop（最热门家居话题）
    性格坚韧 → will of granite / set in granite → iron will / rock-solid
    冷酷无情 → heart of granite / face like granite
    不可改变 → etched in granite / carved in granite（像刻在石头上的决定）
