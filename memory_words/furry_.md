# furry
furry 这个词在英语里有几种完全不同的含义和语境，使用时要特别注意语境，不然很容易误会或尴尬。下面按主要含义给你分类整理同义词、强联想词和地道表达。
1. furry（形容词）—— 毛茸茸的、覆盖着毛的（最中性、最日常用法）
核心同义词（高频、通用）：
fluffy
fuzzy
hairy
shaggy
woolly
downy
fleecy
plush
soft-furred
velvety

强联想词 / 常见搭配：
furry blanket / throw / rug（毛绒毯子）
furry slippers / boots / coat（毛绒拖鞋/靴子/大衣）
furry animal / creature / pet（毛茸茸的动物/宠物）
furry tail / ears / paws（毛茸茸的尾巴/耳朵/爪子）
furry texture / feel（毛绒绒的手感）

地道表达：
This sweater is so furry and warm!（这件毛衣超毛绒超暖和！）
The kitten's paws are incredibly furry.（小猫的爪子毛茸茸的超可爱。）
I love anything furry — blankets, pillows, you name it.（我超爱一切毛绒的东西——毯子、枕头啥都行。）
Her new dog is all black and super furry.（她新养的狗全身黑毛，毛超级多。）

2. furry（名词/亚文化）—— 兽人/拟人动物爱好者（Furry Fandom，次文化圈子）
这是互联网时代最知名的含义，指喜欢拟人化动物角色（anthropomorphic animals）的人群，常涉及画画、cosplay、fursuit（毛绒玩偶装）等。
核心同义词 / 圈内常用称呼：
fursona owner（拥有自己兽设的人）
furry fan / furfan
anthro（anthropomorphic 的缩写）
scalie（喜欢爬行动物拟人化的分支）
kemono（日式兽人风格）
suiters / fursuiters（穿毛绒玩偶服的人）

强联想词 / 圈内高频词汇：
fursona（自己的兽人角色设定）
fursuit（全包毛绒玩偶服）
con / furcon（furry大会，如 Anthrocon、FWA）
badge（大会名牌，通常画有 fursona）
yiff（圈内俚语，指成人向兽人内容，相当于“做爱”的委婉说法）
paws / murr / feral（不同风格偏好）
artist alley（大会艺术家摊位区）
headcanon / species / sona（角色头设、物种、兽设）

地道表达 & 圈内说法（非常口语化）：
“Are you a furry?” / “Yeah, I'm in the fandom.”（你是furry吗？——嗯，我是这个圈的。）
“What's your fursona?”（你的兽设是什么？）
“I just commissioned a new ref sheet for my sona.”（我刚委托画师给我兽设画了新的参考图。）
“I'm heading to Anthrocon next month — super excited to wear my new suit!”（下个月去Anthrocon——超期待穿我的新fursuit！）
“No yiff in the main chat, please.”（主聊天室别发成人内容。）
“He's a total paws guy.”（他超级喜欢爪子元素。）

3. 其他小众 / 俚语用法

furry porn / yiff → adult furry content（成人兽人内容）
get furry → 变得毛茸茸（很少用，但有时开玩笑说）
furry logic → 圈内自嘲的“非人类逻辑”（荒诞但可爱的思维方式）

记忆小联想链

物理毛绒 → fluffy / fuzzy / shaggy → furry blanket / slippers
可爱动物 → furry kitten / puppy → 日常夸萌
亚文化 → fursona + fursuit + con → “I'm a furry” / “What's your sona?”
成人向 → yiff（圈内黑话，外面别随便说）

总结一句话区分：
说“this cat is so furry” → 安全、可爱
说“I'm a furry” → 立刻进入兽人亚文化语境
