# handtruck

hand truck 的词义总结（英式/美式通用）：

主要含义（名词）：一种 L 形的手推车（handcart），有两个大轮子、一个底部小平台和长手柄，用于搬运重物（如箱子、家具、货物）。使用时将平台插入货物下方，向后倾斜，利用杠杆原理滚动搬运。
也常称为 two-wheeler（两轮车）。
注意：常与 dolly 混淆，但严格区分：
hand truck：两个轮子、直立 L 形、可倾斜杠杆式。
dolly：通常四个小轮子、平平台、无高把手（更像低矮移动平台，常用于家具或设备）。


拼写：常用作两个词 hand truck，也可连写成 handtruck（较非正式）。
1. 同义词（Synonyms）
根据地区和语境，最常见的有：

dolly（美式最流行，常互换使用，但技术上略有区别）
hand trolley / trolley（英式/澳式常见）
sack truck / sack barrow（英式，强调搬运麻袋/货物）
bag barrow（新西兰/部分英式）
stack truck
box cart / two wheeler
luggage barrow（行李用）
trundler（较少见）
其他广义：cart、pushcart、handcart、wheelbarrow（独轮手推车，有区别）

反义词（Antonyms）：无直接对应，通常用 forklift（叉车）、pallet jack（托盘车）对比（机械化 vs. 手动）。
语境区别：在仓库/搬家场景，hand truck 和 dolly 常被 interchangeably 使用，但专业场合会区分轮子数量和设计。
2. 联想词 / 相关词汇（Related / Associated Words）

类型：convertible hand truck（可转换式，可变平板车或手推车）、appliance dolly（家电专用）、heavy-duty hand truck（重型）、folding hand truck（折叠式）。
部件：handles（把手）、nose plate / toe plate（底部平台）、wheels（轮子，常有充气或实心）、frame（框架）。
相关工具：pallet jack（手动托盘车）、forklift（叉车）、moving straps（搬家带）、creeper（维修滑板车）。
场景：warehouse（仓库）、moving day（搬家日）、delivery（送货）、retail stockroom（零售仓库）。
词源：hand（手动） + truck（小车/推车），强调人力推动。
文化联想：搬家公司工人、仓库物流、DIY 搬重物、老式体力劳动工具。常出现在搬家视频、仓库安全培训中。

3. 地道表达 / 自然用法（Collocations & Idiomatic Expressions）

常见搭配：
use / load / push / tilt a hand truck
move / haul / transport boxes / furniture on a hand truck
heavy-duty / folding / convertible hand truck
grab the hand truck（口语：拿手推车来）

地道例句（听起来自然）：
“We used a hand truck to move all the heavy boxes into the new apartment.”（我们用手推车把所有重箱子搬进了新公寓。）
“Make sure you tilt the hand truck back before you start rolling it.”（开始推之前，先把推车向后倾斜。）
“The delivery guy brought a dolly — I mean a hand truck — to unload the fridge.”（送货员拿了个手推车来卸冰箱。）
“In the warehouse, everyone calls it a sack truck, but here we just say hand truck.”（在仓库里大家都叫它 sack truck，但这里我们就说 hand truck。）
“Don’t try to carry that filing cabinet by hand — get the hand truck.”（别用手抬那个文件柜，去拿手推车。）
“I bought a folding hand truck for moving stuff around the garage.”（我买了个折叠手推车，在车库里搬东西用。）

自然替代表达（日常更常用）：
“Let’s use the dolly to move this.”
“Grab the two-wheeler for those crates.”
“We need something with wheels to haul these boxes.”


在美式英语中，“dolly” 非常流行（即使严格区分时也常混用）；在英式中 “sack truck” 或 “trolley” 更常见。
4. 发音（Pronunciation）

美式 & 英式：/ˈhænd ˌtrʌk/ （HAND truk）
听感：两个词连读，重音主要在 “hand”，第二个 “truck” 较轻但清晰。/æ/ 如 cat，/ʌ/ 如 but。
连写 handtruck 时仍发同样音。


小贴士：自然连读，听起来像 “han-truck”。推荐 Cambridge 或 YouGlish 搜索 “hand truck” 听真实用法（如搬家或仓库视频）。
5. 拼写 / 发音易混词（Confusables）

拼写：
hand truck（最标准，两个词）
handtruck（连写，也可接受，尤其在产品名中）
常见错误：hand-truck（带连字符，较少）、handtruck（漏空格）。

易混词：
hand truck vs. dolly：最常见混淆！很多人（尤其美式）把两者混用，但技术上 dolly 多为四轮平台。
hand truck vs. pallet jack / pallet truck（托盘车，有液压升降叉子，用于托盘）。
hand truck vs. wheelbarrow（独轮手推车，用于园艺/建筑，形状不同）。
hand truck vs. truck（大卡车，完全不同）。


记忆技巧：Hand（手动） + truck（小车）→ 人力 L 形推车。联想“用手推（hand）的 truck（车）搬重物”。区分 dolly：dolly 像“洋娃娃”小巧四轮平台；hand truck 像“手柄长”的两轮杠杆车。
文化/实用小注：在美国搬家公司和仓库中极其常见，常被戏称为“the poor man’s forklift”。很多产品（如 Costco、Home Depot）标为 “hand truck” 或 “appliance hand truck”。安全提示：重载时要小心倾斜和平衡。
