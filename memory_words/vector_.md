# vector

[[victor]] n.胜利者
Vector [ˈvektə] 是一个横跨数学、物理、生物和计算机图形学的核心术语。它的核心逻辑是**“带有方向的量”或“携带者”**。

1. 同义词 (Synonyms)
由于 Vector 极其专业，其同义词通常取决于具体的应用领域：

关于“数学/物理” (Directional quantity):

Trajectory: 弹道、轨道（侧重于运动矢量）。

Heading: 航向（导航中的矢量）。

关于“生物/医学” (Carrier/Transmitter):

Carrier: 携带者（如蚊子是疟疾的 carrier）。

Transmitter: 传播者。

Vehicle: 媒介、载体（如基因治疗中的 viral vehicle）。

关于“设计/计算机” (Graphics):

Path: 路径（矢量绘图中的核心概念）。

Scalable graphics: 可缩放图形。

2. 领域联想与地道表达 (Domain & Idioms)
计算机图形学 (Graphics):

Vector graphics: 矢量图。与 Raster/Bitmap（位图）相对，矢量图无论放大多少倍都不会失真。

编程与数据 (Programming):

Vector array: 向量数组（在 C++ 或 Java 中常指可变长度的列表）。

生物医学 (Biomedicine):

Viral vector: 病毒载体（用于疫苗开发，如新冠疫苗的技术路径之一）。

地道短语:

A vector for change: 改变的媒介/动力。

Attack vector: [网络安全] 攻击途径。指黑客进入系统的具体方式（如通过 SSH 漏洞）。

3. 易混词与发音陷阱 (Confusing Words)
单词	发音	含义	区分技巧
Vector	[ˈvektə]	矢量/载体	结尾是 -tor。
Victor	[ˈvɪktə]	胜利者	开头是 Vi-，联想 Victory。
Sector	[ˈsektə]	部门/扇区	开头是 S-，联想 Section。
Factor	[ˈfæktə]	因素/因子	联想 Factory（工厂里有很多生产因素）。
4. 单词变形 (Word Inflections)
Vector 主要是名词，但在某些专业语境下也可以作为动词使用。

词性	单词	用法示例
名词 (Noun)	vector	The force is represented by a vector.
复数 (Plural)	vectors	Multiple vectors of infection were identified.
形容词 (Adj.)	vectorial	Vectorial addition. (矢量加法)
动词 (Verb)	vector	The plane was vectored toward the airport. (引导航向)
动词变位 (Vector):

第三人称单数: vectors

过去式/过去分词: vectored

现在分词: vectoring

5. 技术应用小贴士
在你的 Linux 或 WSL2 研究中，你可能会频繁遇到 "Interrupt Vector"（中断向量）。这是操作系统底层的一个概念，指的是当硬件发出请求时，处理器跳转去执行特定程序的内存地址。

如果你在做 UI 设计或前端开发，提到图像不模糊，一定要用 "It's a vector file" 而不是 It's a high-quality picture，这样显得更专业。

6. 记忆钩子
想象一个带有箭头的线段：

箭头代表方向（Direction）。

线段长短代表大小（Magnitude）。
这二者合一，就是 Vector。
