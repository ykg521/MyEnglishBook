# knight

**knight** 这个词在英语里有几种主要含义：
1. **中世纪骑士**（mounted [[warrior]] in [[armor]]，贵族战士）
2. **英国爵士**（titled Sir，骑士头衔）
3. **比喻英雄/救星**（尤其是 **knight in [[shining]] armor** 这个超级常见的习语）
4. **国际象棋里的马**（[[chess]] [[piece]]）

下面按含义给你整理**同义词**（synonyms）、**强联想词**（associated words）和**地道表达 / 习语**（idiomatic / natural expressions），方便你记忆和地道使用。

### 1. knight（中世纪骑士 / 武士）

**核心同义词**（最常用）：
- [[cavalier]]（骑士、绅士风度）
- [[chevalier]]（法语借词，骑士，常带浪漫感）
- [[gallant]]（勇敢绅士、骑士）
- [[paladin]]（圣骑士、英雄骑士）
- [[warrior]]（战士，更广义）
- [[horseman]] / [[equestrian]]（骑手、骑士）
- [[champion]]（冠军、捍卫者）
- banneret（旗骑士，低阶骑士）

**强联想词 / 相关词汇**：
- [[armor]]（盔甲）
- [[lance]]（长矛）
- [[sword]]（剑）
- [[shield]]（盾牌）
- [[joust]] / [[jousting]]（马上比武）
- [[chivalry]]（骑士精神、侠义）
- [[quest]]（骑士冒险任务）
- code of honor（荣誉准则）
- [[squire]]（侍从）
- [[castle]] / [[fortress]]（城堡）
- [[crusade]]（十字军东征）
- Templar（圣殿骑士）

**地道表达**：
- knight-[[errant]]（游侠骑士，四处冒险的骑士，像堂吉诃德）
- knight in shining armor（闪亮的铠甲骑士 → 最经典习语！）
- white knight（白骑士 → 商业/金融语境：拯救公司的人；也指英雄救美）
- dark knight（黑暗骑士 → 像蝙蝠侠，神秘/黑暗英雄）

### 2. knight（英国爵士头衔，Sir）

**常见同义词 / 相关**：
- Sir（直接称呼）
- baronet（准男爵）
- lord（勋爵，更高阶）
- peer（贵族）
- dame（女性对应，女爵士）

**地道表达**：
- be knighted（被封为骑士） e.g. He was knighted for his services to music.（他因对音乐的贡献被封爵。）
- Sir Elton John（埃尔顿·约翰爵士）

### 3. knight in shining armor（英雄救美 / 救星）

**同义词 / 替代表达**（超级高频习语）：
- hero（英雄）
- [[savior]] / saviour（救世主）
- [[rescuer]]（救援者）
- [[protector]]（保护者）
- [[guardian]]（守护者）
- [[champion]]（捍卫者）
- Sir Galahad（加拉哈德爵士 → 纯洁无暇的完美骑士）
- white knight（白骑士）
- guardian angel（守护天使）
- tower of strength（力量之塔，靠谱的支柱）
- good Samaritan（好撒玛利亚人，好心帮忙的人）

**地道用法 & 场景**：
- He's my knight in shining armor.（他是我闪亮的救星 / 英雄。→ 浪漫/感激时用）
- She's still waiting for her knight in shining armor.（她还在等她的白马王子。→ 略带讽刺或梦幻）
- Don't wait for a knight in shining armor; save yourself!（别等英雄来救你，自己动手！→ 现代女权/独立语境）
- He rode in like a knight in shining armor and fixed everything.（他像英雄一样出现，把一切搞定。）

### 4. 其他联想 & 现代用法
- chess knight（象棋里的马） → the knight moves in an L-shape（马走日字）
- Dark Knight（黑暗骑士 → 蝙蝠侠电影系列）
- knight of the realm（王国骑士）
- hedge knight（雇佣骑士，低阶/流浪骑士，常出现在奇幻小说如《冰与火之歌》）

### 记忆小联想链
- 中世纪画面 → armor + sword + horse → cavalier / chevalier / paladin
- 浪漫英雄 → knight in shining armor → hero / rescuer / Sir Galahad
- 现代吐槽 → waiting for a knight in shining armor（等白马王子，等救星）
- 反讽 → rusty knight（生锈的骑士，没那么完美）或 tarnished knight（名声有污点的骑士）
