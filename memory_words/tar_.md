# tar

[[tar]] 这个词在英语里主要有两种核心用法（按常见度排序）：

名词：焦油、沥青、柏油（黑乎乎、黏黏的物质，用于铺路、防水等）
动词：涂焦油、刷沥青

此外还有几个超级经典的习语和比喻用法。
下面按含义给你整理同义词、强联想词和地道表达。
1. tar（名词）—— 焦油、沥青（最常见用法）
核心同义词（高频、通用）：

pitch（最接近，尤其木焦油或船用）
bitumen（正式/工程用词，尤指道路沥青）
asphalt（美式英语中最常见的道路铺设材料）
creosote（木焦油，常用于木材防腐）
blacktop（美式口语，指铺好的沥青路面）

强联想词 / 常见搭配：

road tar / highway tar（道路沥青）
coal tar（煤焦油）
cigarette tar（香烟焦油 → 健康警告常用）
tar pit（沥青坑，如美国La Brea Tar Pits）
roofing tar（屋顶防水焦油）
sticky tar / hot tar（热沥青，超级黏）
tar and gravel（碎石沥青路面）

地道表达：

The road was covered in fresh tar — it was still sticky.（路刚铺了新沥青，还黏糊糊的。）
Cigarette tar is one of the main causes of lung cancer.（香烟焦油是导致肺癌的主要原因之一。）
The dinosaurs got trapped in the tar pits.（恐龙被困在了沥青坑里。——经典科普句）
Don’t step on the tar or your shoes will be ruined!（别踩到沥青，不然鞋就毁了！）

2. tar（动词）—— 涂焦油、刷沥青
核心同义词：

coat with tar
asphalt（动词用法：铺沥青）
pitch（用沥青涂）
seal（密封，常说 tar and seal）

地道表达：

They’re tarring the roof this week.（他们这周在给屋顶刷防水沥青。）
The workers tarred the driveway yesterday.（工人昨天把车道铺了沥青。）

3. 经典习语 & 比喻用法（最实用！）

tar and feather（涂焦油粘羽毛 → 旧时公众羞辱惩罚，现在多比喻严厉惩罚）
e.g. If he keeps lying, they’ll tar and feather him on social media.（他再继续撒谎，大家会在网上把他“涂焦油粘羽毛”。）
tar sb with the same brush（用同一把刷子刷 → 一竿子打翻一船人，把一类人全归为坏的）
e.g. Don’t tar all politicians with the same brush — some are honest.（别把所有政客一棍子打死，有些还是诚实的。）
Jack Tar（老水手、英国水兵的昵称，历史用法）
e.g. The old Jack Tar told stories of life at sea.（那个老水手讲了很多海上生活的故事。）
tar baby（黏人的麻烦、越陷越深的困境 → 来自美国民间故事）
e.g. The project became a real tar baby — every fix made it worse.（这个项目成了个“焦油娃娃”，越修越麻烦。）

4. 记忆小联想链（超级好记）

黏 + 黑 + 热 → sticky black tar → pitch / bitumen / asphalt（铺路画面）
惩罚画面 → tar and feather（历史上真的把人涂焦油粘鸡毛游街）
一竿子打死 → tar with the same brush（像用同一把刷子全刷黑）
陷阱比喻 → tar pit / tar baby（越挣扎越黏，越陷越深）
健康警告 → cigarette tar（烟盒上最常见的警告词）

一句话总结：
tar = “又黑又黏的焦油”，最地道的用法是 tar and feather（严厉惩罚）和 tar with the same brush（一竿子打翻一船人），日常最常听到的是铺路或香烟焦油。
