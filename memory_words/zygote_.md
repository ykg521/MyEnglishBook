# zygote
zygote 这个词是生物学（尤其是发育生物学和生殖医学）里的专业术语，意思是受精卵：精子（sperm）和卵子（egg / ovum）结合后形成的第一个单细胞，是新生命发育的起点。
它在日常英语中出现频率不高，主要用于科学、医学、IVF（体外受精）语境，几乎没有日常口语习语，但有固定学术表达。下面按类别整理同义词、强联想词和地道表达。
1. 同义词（Synonyms）
最直接 / 高频替换（科学写作和口语中最常用）：

fertilized egg（受精卵，最通俗、最常用）
fertilized ovum（受精卵，更正式的拉丁词）
one-celled embryo / single-cell embryo（单细胞胚胎）
zygotic cell（受精卵细胞，学术味更重）

稍正式 / 技术性同义：

zygote cell
initial zygote
diploid cell（二倍体细胞，强调染色体数量，但不完全等同）

注意：

zygote → 严格指“刚受精的单细胞阶段”
embryo（胚胎）→ zygote分裂几次后（通常到囊胚阶段）才叫embryo
所以不要随意互换。

2. 强联想词 / 常见搭配（Collocations）
这些是科学论文、科普文章、医生对话里最常出现的组合：

zygote formation / zygote formation during fertilization（受精过程中形成受精卵）
zygote cleavage / zygote divides（受精卵卵裂 / 分裂）
zygote development / zygote stage（受精卵发育阶段）
zygote implantation（受精卵着床）
human zygote / mammalian zygote（人类/哺乳动物受精卵）
in vitro zygote / IVF zygote（体外受精的受精卵）
zygote transfer / zygote culture（受精卵移植 / 培养）
from zygote to blastocyst（从受精卵到囊胚）
zygote genome activation（受精卵基因组激活，ZGA，学术高频词）

3. 地道表达 & 真实场景用法

The zygote begins to divide rapidly right after fertilization.
（受精后，受精卵会迅速开始分裂。——最标准科普句）
In IVF, doctors monitor the zygote for 3–5 days before transferring it to the uterus.
（在体外受精中，医生会培养受精卵3–5天后再移植到子宫。）
The zygote is the very first stage of a new human life.
（受精卵是新人类生命的最初阶段。——伦理/科普讨论中常见）
After cleavage, the zygote becomes a morula, then a blastocyst.
（经过卵裂后，受精卵变成桑椹胚，再变成囊胚。）
The fertilized egg (zygote) implants in the uterine wall about 6–10 days later.
（受精卵大约6–10天后会在子宫壁着床。）
Scientists can now edit the zygote’s DNA using CRISPR.
（科学家现在可以用CRISPR编辑受精卵的DNA。）
From a single zygote to a full-grown organism — that’s the miracle of development.
（从一个受精卵到一个完整生物体——这就是发育的奇迹。）

口语/科普简化版（非专业场合）：

“The sperm and egg fuse to create a zygote — basically the first cell of the baby.”
“In the lab, we watch the zygote grow into an embryo.”

4. 记忆小联想链（超级好记）

词源：来自希腊语 zygon（yoke = 轭），意思是“两个配子像被轭套在一起结合” → zygote = “结合后的第一个细胞”
过程链：
gamete（配子）→ fertilization（受精）→ zygote（受精卵）→ cleavage → morula → blastocyst（囊胚）→ embryo → fetus
画面联想：一个微小的单细胞，里面已经包含全部遗传信息，像“生命的起点胶囊”
现代场景：IVF诊所里最常听到的词就是 “How many zygotes did we get?”

一句话总结：
zygote = “生命最开始的那个单细胞”，英语里几乎只用于科学语境，日常最自然的说法就是 fertilized egg，而 zygote 更常出现在论文、IVF报告和发育生物学课上。
