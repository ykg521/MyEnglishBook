# batten
batten 这个词在英语里有几种主要含义，常见程度从高到低排序：

最常见习语：batten down the hatches（封舱准备、做好应对麻烦的准备）—— nautical 起源，超级高频比喻用法
动词：用木条固定、钉牢、加固（尤其是盖住舱口或建筑）
名词：木条、压条、板条（建筑/舞台用）
较少见 / 旧式：batten on / upon（靠…发财、寄生、靠别人养活自己，带负面意味）

下面按含义给你整理同义词、强联想词和地道表达。
1. batten down the hatches（习语）—— 准备迎接风暴/麻烦、严阵以待
核心同义词 / 替代表达（最常用）：
brace yourself / [[brace]] for impact
prepare for the worst
get ready for trouble
hunker down
buckle up (for a rough ride)
batten down (简写版)
dig in / dig your heels in
gird your loins (稍正式/幽默)
batten the hatches (变体)

强联想词 / 搭配：
storm / crisis / recession / deadline / bad news coming
prepare / ready / fortify / secure
tough times ahead / rough seas / incoming trouble

地道表达 & 真实场景（native speaker 超爱用）：
Batten down the hatches — the boss is in a foul mood today!（准备好，大老板今天心情超差！）
With the market crash coming, everyone’s battening down the hatches.（股市要崩，大家都在做最坏准备。）
Storm’s approaching — batten down the hatches!（暴风雨要来了——封好舱！→ 字面+比喻）
We need to batten down the hatches and cut costs immediately.（得赶紧勒紧裤腰带缩减开支。）
Kids are coming home for the holidays — batten down the hatches!（孩子们要回家过节了——家里要乱套了，准备好！→ 幽默）

2. batten（动词）—— 用木条固定、钉牢、加固
核心同义词（高频）：
[[fasten]]
[[secure]]
board up
nail down
clamp down
tie down
[[barricade]]
fix / attach
strengthen / reinforce

强联想词：
hatches / tarpaulin / canvas（船上盖布）
windows / doors（暴风雨前钉木板）
stage lights / scenery（剧院里的 batten = 吊杆）

地道表达：
They battened down the hatches before the typhoon hit.（台风来前他们把舱口封牢了。）
Batten the tarpaulin securely so it doesn’t blow away.（把防水布用木条钉牢，别被风刮走。）
The windows were battened up against the hurricane.（窗户用木板钉死防飓风。）

3. batten（名词）—— 木条、压缝条、舞台吊杆
常见同义词：
[[slat]]
[[lath]]
strip (of wood)
board
[[cleat]]
pipe batten (舞台灯光吊杆)

强联想词：
roofing / tiling / flooring（建筑用）
hatch batten（船用）
lighting batten / fly batten（剧院）

4. batten on / upon（靠…发财、寄生）—— 较旧/文学用法
同义词：
thrive on / at the expense of
feed off
sponge off
live off
exploit / profit from
fatten on

地道表达（现在少见，但文学/批评中出现）：
Politicians batten on public funds.（政客靠公款养肥自己。→ 负面）
He battens on his rich relatives.（他靠有钱的亲戚过寄生生活。）

记忆小联想链
海上风暴 → seal the hatches with battens → batten down the hatches → 准备大麻烦！
建筑/船 → nail / fasten / secure with wooden strips → batten
寄生虫 → feed / thrive / fatten on others → batten on
最强画面 → 暴风雨前水手钉木条封舱 → “Batten down the hatches!”（经典喊话）
