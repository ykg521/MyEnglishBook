# tunnel 单词笔记
英 /ˈtʌnl/　美 /ˈtʌnl/
n. 隧道；地道；坑道
v. 挖隧道；开凿地道
一、同义词
名词（隧道 / 通道类）
underpass 地下通道；下穿隧道
subway（英）地下隧道；（美）地铁
underground passage 地下通道
culvert 涵洞；排水隧道
shaft 竖井；坑道
cavern 大洞穴；洞窟
duct 管道；风道
conduit 导管；管线通道
动词（挖隧道 / 打通）
excavate 挖掘；开凿
dig 挖
burrow 掘洞；打洞（动物 / 人）
mine 采矿；挖地道

五、小辨析
tunnel：人工开凿的长形地下通道（交通、排水等）
cave / cavern：天然形成的洞穴
underpass：道路下穿式小型地下通道
subway：英式多指行人隧道，美式指地铁系统
