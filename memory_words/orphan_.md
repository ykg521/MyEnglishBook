# orphan

orphan（名词/形容词/动词）主要含义：

名词：失去双亲（或一方）的孤儿（a child whose parents are both dead, or sometimes one parent）。
形容词：孤儿的、失去父母的（orphaned）。
动词：使成为孤儿（to cause someone to become an orphan）。
比喻/专业用法：缺乏支持或照顾的事物（如 orphan project / orphan drug / orphan page in publishing）。

它常带有悲伤、孤独、无依无靠的情感色彩，在文学、新闻和社会话题中频繁出现。
发音（Pronunciation）

美式英语：/ˈɔːr.fən/（OR-fən）——重音在第一个音节，“or”像 “or” 或 “for”，第二个音节弱读。
英式英语：/ˈɔː.fən/（AW-fən）——类似美式，但第一个元音稍长。
复数 orphans：/ˈɔːr.fənz/。
过去式/形容词 orphaned：/ˈɔːr.fənd/。

常见错误发音：不要把重音放在第二个音节（不要读成 or-FAN）。
同义词（Synonyms）
1. 核心含义（child without parents）

foundling（弃婴、被遗弃的婴儿，最接近）
waif（流浪儿、无家可归的孩子，常带可怜感）
stray（流浪儿）
urchin（街头顽童，常指脏兮兮的孤儿）
ragamuffin（衣衫褴褛的孩子）
abandoned child / parentless child
motherless / fatherless（单亲丧亡）

2. 形容词形式（orphaned）

bereaved（丧失亲人的）
parentless / unparented
adopted（被收养的，但不完全同义）

3. 比喻用法（lacking support）

abandoned / neglected / unsupported
orphan project / orphan drug（孤儿药，指罕见病药物）

反义词（Antonyms）：

adoptee / adopted child
parented / with parents
foster child（寄养儿童，不完全反义）

联想词 & 常见搭配（Collocations）

orphan child / orphan boy / orphan girl
orphaned children / orphaned by the war / accident
orphanage（孤儿院，最强关联词）
adopt an orphan
orphan drug（医药：孤儿药）
orphan drug说明：孤儿药是指用于治疗罕见疾病的药物，这类疾病患者数量较少，因此制药公司通常缺乏开发这类药物的动力。政府为了鼓励药物开发，通常会提供一些激励措施，例如税收减免或市场独占权等。
orphan drug近义词：rare disease medicine(罕见病药物), niche drug(利基药物)

反义词：blockbuster drug(重磅药物), common drug(常用药物)
orphan project / orphan page（出版/技术：无人维护的项目或页面）
half-orphan（失去单亲）
Dickensian orphan（狄更斯式孤儿，文学经典形象）

例句：

She was orphaned at the age of five.
The war left thousands of orphaned children.
This software is an orphan project with no active developers.

地道表达 & 习语（Natural Expressions）
英语母语者常用以下自然、生动的说法：

Orphaned by（因…成为孤儿）
“Many children were orphaned by the earthquake.”
Take in an orphan / Adopt an orphan
“The couple decided to adopt an orphan from overseas.”
Orphan drug / orphan disease（专业但常见于新闻）
“Developing orphan drugs for rare diseases is very expensive.”
Success has many fathers, but failure is an orphan.（经典谚语）
意思：成功人人争功，失败却无人认领。

其他口语/文学用法：

“He grew up as a poor orphan in a Dickens novel.”（狄更斯式孤儿形象）
“The project became an orphan after the funding was cut.”
“She felt like an emotional orphan after losing her parents.”（情感上的孤儿）
“Don’t leave the document as an orphan at the bottom of the page.”（排版术语：孤行）

文学经典联想：

Oliver Twist、Annie（Orphan Annie）、Jane Eyre 等作品中的孤儿形象，常象征坚韧、苦难与希望。

拼写 & 易混词（Spelling & Confusable Words）

正确拼写：orphan（o-r-p-h-a-n）
常见错误：orphin、orfan、orphane（多加 e）。

易混词：

orphan vs often（发音最接近的混淆）
Orphan：/ˈɔːr.fən/（孤儿）。
Often：/ˈɔː.fən/ 或 /ˈɒf.ən/（经常）。
两者发音非常相似，非母语者容易听混，尤其在快速说话时。

orphan vs orphanage
Orphan：孤儿（人）。
Orphanage：孤儿院（地方）。

orphan vs orphan（比喻 vs 字面）
字面：失去父母的孩子。
比喻：无人支持的事物（orphan project）——现代商务/科技中很常见。

orphaned vs adopted
Orphaned：成为孤儿（失去父母）。
Adopted：被收养（获得新父母）。


其他小提醒：

在科技领域，“orphan” 常指“孤立、无关联”的东西（如 orphan files）。
无明显同音异义词（homophone），但发音与 “often” 高度相似。

中文对应地道说法（参考）

孤儿、孤儿院（orphanage）
弃婴（foundling）
失去父母的（orphaned）
孤儿药（orphan drug）
狄更斯式孤儿（可怜而坚强的孤儿形象）

小结使用建议：

日常/新闻 → orphaned children、orphanage。
专业/商业 → orphan project、orphan drug（非常地道）。
想显得文学化 → 联想狄更斯或谚语 “failure is an orphan”。
