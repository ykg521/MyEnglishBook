# flannel

flannel 是一个典型的英美差异极大的多义词，主要含义如下（按使用频率）：

布料：柔软、有绒毛的织物（传统羊毛，现在多棉质），用于保暖衣服。
英式/澳式/新西兰：洗脸巾 / 毛巾布（= 美式 washcloth）。
英式俚语（非正式）：废话、空话、奉承话、回避真相的圆滑言语（= nonsense, flattery, hedging）。
美式：flannel shirt（法兰绒衬衫，常带格子图案）。
同义词（Synonyms）
1. 布料 / 织物（fabric sense）
flannelette（棉质仿法兰绒，最接近的常见替换）
brushed cotton（刷毛棉）
wool flannel / cotton flannel
fleece（有时混用，但实际不同，见易混词）
plaid（常指图案，但不是布料本身）
2. 英式洗脸巾
washcloth（美式）
facecloth（通用）
face flannel（强调）
3. 英式俚语（废话 / 奉承）
nonsense / rubbish / baloney / claptrap
flattery / sweet talk / blarney / soft soap
waffle（含糊其辞）
hedging / prevarication / equivocation（回避）
crock (of shit)（粗口版）
动词形式（to flannel someone）：奉承、哄骗 → flatter、butter up、sweet-talk。

联想词 & 常见搭配（Collocations）
flannel shirt（美式最常见：格子法兰绒衬衫， lumberjack / hipster 风格）
flannel sheets / flannel pajamas / flannel nightgown（保暖床上用品）
grey flannel suit（经典比喻：20世纪中叶公司职员的保守西装，源自小说/电影《The Man in the Grey Flannel Suit》）
flannel trousers / flannels（旧式羊毛法兰绒裤）
cotton flannel / brushed flannel
talk flannel / a load of flannel（英式：说废话）
pure flannel / don't talk flannel（别胡说 / 别绕弯子）
例句：

I love wearing a red plaid flannel shirt in fall.（美式）
Pass me the flannel so I can wash my face.（英式）
Stop talking flannel and give me a straight answer!（英式俚语）
地道表达 & 习语（Natural Expressions）
Talk flannel（英式，非常地道）：说空话、废话、圆滑话
“The politician was just talking flannel as usual.”
A load of old flannel（英式）：一大堆废话
“That excuse is a load of old flannel.”
Flannel shirt season（美式口语）：秋冬穿法兰绒衬衫的季节（常带轻松、复古感）
Grey flannel（比喻）：保守的企业文化
“He escaped the grey flannel world of corporate America.”
Buffalo plaid flannel（美式时尚）：经典红黑格子法兰绒（lumberjack look）
其他生动用法：

“He’s in his flannel pajamas watching football.”（美式，舒适居家）
“Don’t give me any flannel — just tell me the truth.”（英式，直率要求）
易混词（Confusable Words）及区别（这是 flannel 最容易出错的地方！）
flannel vs flannelette（最常见混淆）
Flannel：原指羊毛或高质量刷毛织物（可双面刷毛）。
Flannelette：棉质单面或双面刷毛的廉价仿制品（现在很多“flannel shirt”其实是 flannelette）。
母语者常互换，但严格来说 flannelette 更轻、更便宜。
flannel vs fleece（超级容易搞混）
Flannel：梭织（woven）布料，通常棉或羊毛，有轻微绒毛，手感像“软磨砂”。
Fleece：针织（knit）合成纤维（聚酯），更厚、更保暖、像毛绒玩具，不易缩水。
例：户外抓绒衣 = fleece；秋季格子衬衫 = flannel。
flannel（英式） vs washcloth（美式）
同一个东西：在英国叫 flannel，在美国叫 washcloth 或 facecloth。
如果美国人说 “I need a flannel”，英国人会以为是衬衫；反之英国人说 “pass the flannel”，美国人会困惑。
flannel vs plaid / check
Flannel 是布料材质。
Plaid / check 是图案（格子）。很多 flannel shirts 是 plaid，但不是同义词。
英美差异总结（最重要！）：

美式：flannel = 保暖布料 / 法兰绒衬衫（日常最常见）。
英式：flannel = 洗脸小毛巾（日常） + 俚语废话；布料则说 flannel fabric / flannelette。
中文对应地道说法（参考）
布料：法兰绒、法兰绒衬衫（格子衬衫）
英式毛巾：洗脸巾、小方巾
俚语：废话、空话、瞎扯、拍马屁、绕弯子的话
小结使用建议：

在美国/加拿大 → 多说 flannel shirt（秋冬必备词）。
在英国/澳洲 → flannel = 洗脸巾；想说布料最好说 flannel shirt 或 brushed cotton；说废话就用 talk flannel（非常地道）。
想避免混淆 → 描述材质时加 cotton flannel 或 wool flannel。
