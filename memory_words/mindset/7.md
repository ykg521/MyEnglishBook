### 工程素养：
- rigor (严谨/严厉 - 高代码质量的核心要求)
- dissemination (传播/散布 - 如 Data dissemination 数据分发)












- anesthesiologist (麻醉师) + femur (股骨) + spasm (痉挛)：想象一个骨科手术场景，这几个词就串起来了。
- karaoke (卡拉OK) + sake (清酒) + polka (波尔卡)：一个充满音乐和酒精的聚会场景。
- loophole (漏洞) + compile (编译) + null (空值)：你的日常 Bug 修补工作。
- CPU Spike (CPU 峰值)：在分析音视频解码性能时，监控 spike 是定位 Anr (应用无响应) 的第一步。
- Data Dissemination (数据分发)：在 A/V SDK 中，描述如何将流数据高效地 dissemination 到多个渲染层。
- Rigor (严谨性)：面试时强调你对单元测试的 rigor 追求，是 Senior 职位的加分项。
- Memory Chunk (内存块)：在处理 Native Memory (原生内存) 分配时，你管理的是一个个内存 chunk。
- Oxidation (物理损耗)：在开发 IoT 或嵌入式 A/V 设备时，考虑对硬件 oxidation 的影响，决定了产品的生命周期。
- Cinematic (电影感)：在 2026 年的 A/V 趋势中，如何利用 AI 实现移动端的 cinematic depth of field (电影级景深) 是核心竞争力。
- Lazy Loading (懒加载)：虽然 laziness 听起来是负面的，但在 Android 开发中，合理的 lazy initialization 是内存优化的基石。
- Cradle (底座)：在开发车载 A/V 系统或智能家居 SDK 时，需要考虑设备处于 charging cradle 模式下的低功耗策略。
- Needless (冗余)：Senior 工程师的价值在于通过 code review 砍掉那些 needless complexity (不必要的复杂性)。
- Jingle (音频片段)：在音频处理中，jingle 的淡入淡出（Fade in/out）算法是基础功能。
