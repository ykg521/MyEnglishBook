### 程度、修饰与高级表达

- more (更多的/更大程度地)。
- gave (给/赋予 - give 的过去式)。
- myriad (极大量的)
- enough (足够)
- too (太)
- largest (最大的)。
- excusable (可原谅的)
- oncoming (迎面而来的)
- accompanying (伴随的)
- wholly (完全地)
- quite (相当/很)
- handedly (单手地/独力地)
- nuance (细微差别)：描述 A/V 编码参数或系统性能调优中的微妙差异。
- nonetheless (尽管如此)：高级转折词，等同于 nevertheless。
- concomitant (伴随的/同时发生的)：描述 Bug 产生时的伴随现象。
- akin (相似的/同族的)：This issue is akin to the memory leak we found last week.
- ubiquitous (无处不在的)：形容像 Android 设备这样普及的事物。
- unfathomable (深不可测的/难以理解的)：形容某些“史诗级”的 Legacy code 或复杂的 Bug。
