- avocado (牛油果) + balsamic (香醋) + noodle (面条)：想象一份健康餐，能帮你把这几个生活词串在一起。
- avalanche (雪崩) vs onslaught (突袭)：两者都带有“不可阻挡的冲击”之意，雪崩是自然的，onslaught 多指人为或抽象的。
- sync (同步) + standardisation (标准化)：这是开发中的高频场景，同步代码、统一标准。
- Memory Leak (内存泄漏)：13 年经验的你，排查 Leak 是家常便饭。
- Tilt (倾斜/陀螺仪)：在开发 A/V SDK 的 VR 模式或视频防抖时，处理传感器传回的 tilt angle (倾斜角)。
- ms (毫秒)：在音视频同步中，10ms 的偏移都可能导致 Lip-sync 失败。
- Relentless (不懈)：描述你在解决一个 Heisenbug (难以重现的 Bug) 时那种 relentless 的态度。
- Hustle (忙碌/进取)：在初创公司（Startup）或快速迭代的团队中，这种 hustle culture 非常常见。
- Memory Dump (内存转储)：当 SDK 崩溃（Crash）时，通过分析 dump file
- Critical Path (关键路径)：在 A/V SDK 优化中，缩短渲染流水线的 path 是提升帧率的关键。
- upsurge (激增)：描述开启 4K 录制时，系统功耗和发热量的 upsurge。
- blind spot (盲区)：在进行单元测试（Unit Testing）或代码覆盖率分析时，找出那些逻辑 blind spots。
- droop (掉帧/下降)：描述在低电量模式下，设备由于降频导致的 frame rate droop (帧率下降)。
