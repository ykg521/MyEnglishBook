# ruin
##### 词义
- 英：/ˈruːɪn/； 美：/ˈruːɪn/
- #词性/vt  破坏；毁灭；使破产(或失去地位等)；毁坏；糟蹋
- #词性/n  破坏；毁灭；破产；毁坏；残垣断壁；祸根；失去工作(或地位等)；一无所有；破产(或丢掉工作等)的根源

##### 同义词
1. ruin（动词）—— 破坏、毁坏（最常见用法）
核心同义词（高频、通用）：
destroy
wreck
demolish
devastate
ravage
smash
crush
spoil
damage
ruin (本身最中性)

更严重的毁灭 / 彻底毁掉：
annihilate
obliterate
lay waste (to)
raze
torpedo
undo
dash (hopes)

经济 / 个人层面毁掉（破产、毁前途）：
bankrupt
bust
break
impoverish
wipe out
clean out
bring down

轻度 / 搞砸 / 糟蹋：
spoil
mess up
screw up
botch
mar
blight

常见搭配：
ruin someone's life / career / reputation（毁掉人生/事业/名声）
ruin the surprise / moment / mood（毁掉惊喜/气氛）
ruin a relationship / friendship（毁掉关系）
ruin the economy / country（毁掉经济/国家）
ruin everything（全毁了）

2. ruin（名词）—— 毁灭、废墟、祸根

废墟 / 残骸（古建筑等）：

remnants
wreckage

vestiges

比喻 / 导致毁灭的原因：
bane

4. 记忆小联想链
物理破坏 → wreck / demolish / raze（像推倒大楼）
彻底毁灭 → devastate / annihilate / lay waste（像战争/灾难后）
经济破产 → bankrupt / bust / wipe out（像股市崩盘）
搞砸机会 → spoil / ruin / blow（口语：You just ruined it!）
废墟画面 → ruins / rubble / remnants（古罗马废墟、游戏里的ruins）
导致失败的根源 → bane / downfall / undoing（His pride was his ruin.）
