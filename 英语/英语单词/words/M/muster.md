# muster
##### 词义
- 英：/ˈmʌstə(r)/； 美：/ˈmʌstər/
- #词性/v  集合；召集；聚集；集结(尤指部队)；激起(支持、勇气等)；赶拢(牛、羊)
- #词性/n  聚集的人群；(尤指)集结的兵力
##### 例句
- The regiment commander ordered ( or gave orders for ) an emergency muster .
	- 团长下令紧急集合。
- Go and muster all the men you can find .
	- 去把能找到的人都集合起来。
- We mustered what support we could for the plan .
	- 我们极尽所能为这项计划寻求支持。

##### 同义词
1. muster（动词）—— 集合、召集、聚集
核心同义词（最常用）：
assemble
gather
collect
rally
summon
convene
marshal
mobilize
round up
call together

** muster up / muster the courage / strength 等（鼓起勇气、力量）** 的近义表达：
summon up
rally
gather
scrape together
whip up
call up
work up

2. muster（名词）—— 集合、点名、队伍检阅
常见同义词：
assembly
gathering
roll call
headcount
lineup
inspection
parade
convocation
roundup
congregation

##### 联想词
强联想词 / 常见搭配（collocations & 语境词）
这些词经常和 muster 一起出现，特别有助于理解军事 / 正式 / 鼓起力量的语感：
muster the troops / forces / soldiers（召集部队）
pass muster（合格、过关、达到标准）← 超级常见习语！
muster roll（花名册、点名册）
morning muster（早点名）
muster station（集合点，如船上紧急集合地点）
muster courage / strength / energy / enthusiasm（鼓起勇气/力量/热情）
muster support / votes（争取支持/选票）
fail to muster（没能集合起、凑不够）

记忆小联想链（从军事 → 日常）
军队点名 → muster
→ 集合大家 → assemble / rally
→ 鼓起最后一点力气 → muster up courage
→ 东西凑够标准 → pass muster
→ 没达到标准 → doesn't pass muster
